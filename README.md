## 💡 The Challenge
Replicating a fintech leader's interface like Razorpay is a true test of a developer's attention to detail. I built this clone to push my **Tailwind CSS** skills to the limit—focusing on complex layout structures, overlapping elements, and maintaining a professional, high-trust aesthetic using utility-first principles.

## 🚀 Technical Highlights
- **Utility-First Architecture:** Built the entire complex interface without leaving the HTML, utilizing Tailwind's powerful utility classes for a lightning-fast UI.
- **Complex Grid & Flexbox:** Replicated intricate sections like the "Payment Suite" and "Business Banking" cards using advanced CSS layouts.
- **Responsive Engineering:** Engineered a seamless transition from desktop to mobile, ensuring the professional fintech look remains intact across all devices.
- **Custom Theming:** Configured custom color palettes and spacing scales to match Razorpay’s exact brand identity.

## 🛠 Tech Stack
- **Framework:** Tailwind CSS (v3.x)
- **Structure:** Semantic HTML5
- **Icons & Assets:** Optimized SVG icons for crisp visuals at any scale.
- **Hosting:** GitHub Pages

## 🧠 What I Mastered
This project was a deep-dive into modern frontend workflows:
1. **Precision Styling:** Learning how to achieve pixel-perfect results using Tailwind's spacing and typography scales.
2. **Component Abstraction:** Identifying repeating patterns (like feature cards) and thinking about how to build them modularly.
3. **Performance:** By using Tailwind, I realized how tiny the final CSS bundle can be even for such a visual-heavy site.
